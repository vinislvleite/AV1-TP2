## Como executar o projeto

Siga os passos abaixo para compilar e rodar o projeto:

### Compilar o TypeScript

Abra o terminal e execute:

```bash
cd ./AV1
npx tsc
```

---

### Executar o projeto

Depois da compilação, rode:

```bash
cd ./js/teste
node index.js
```
