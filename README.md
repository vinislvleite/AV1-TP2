## Como executar o projeto

Siga os passos abaixo para compilar e rodar o projeto:

### 1- Clone o repositório

### 2- Compilar o TypeScript

Abra o terminal e execute:

```bash
cd ./AV1
npx tsc
```

---

### 3- Executar o projeto

Depois da compilação, rode:

```bash
cd ./js/teste
node index.js
```
